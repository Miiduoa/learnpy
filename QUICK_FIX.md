# 🔧 Pyodide 載入問題 - 快速修復指南

## 如果一直顯示「正在載入 Python 執行環境...」

### 🚀 立即嘗試的解決方案

#### 1. 使用診斷工具（推薦）
在載入畫面中點擊 **「🔍 診斷問題」** 按鈕，系統會自動檢查：
- 網絡連接
- CDN 可訪問性
- Pyodide 腳本狀態
- 並提供具體的解決建議

#### 2. 檢查瀏覽器控制台
1. 按 `F12` 打開開發者工具
2. 切換到「Console」標籤
3. 查看是否有 `[PYODIDE]` 開頭的錯誤訊息
4. 截圖保存錯誤訊息

#### 3. 強制刷新頁面
- Windows/Linux: `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

#### 4. 清除瀏覽器緩存
1. 按 `Ctrl + Shift + Delete` (Mac: `Cmd + Shift + Delete`)
2. 選擇「緩存的圖片和文件」
3. 點擊「清除數據」
4. 刷新頁面

#### 5. 檢查網絡連接
在瀏覽器控制台（F12）中執行：
```javascript
fetch('https://cdn.jsdelivr.net/pyodide/v0.24.1/full/pyodide.js', { method: 'HEAD' })
  .then(() => console.log('✓ CDN 可訪問'))
  .catch(() => console.error('✗ CDN 無法訪問 - 需要使用 VPN'));
```

如果顯示「✗ CDN 無法訪問」，請使用 VPN。

## 🔍 常見問題和解決方案

### 問題 1: 網絡連接問題
**症狀：** 載入超過 60 秒無響應

**解決方案：**
- 檢查網絡連接是否正常
- 嘗試訪問其他網站確認網絡可用
- 如果使用公司/學校網絡，可能需要 VPN

### 問題 2: CDN 被阻止
**症狀：** 控制台顯示「無法從 ... 載入腳本」

**解決方案：**
- 使用 VPN 切換節點
- 檢查防火牆設置
- 嘗試使用手機熱點

### 問題 3: 瀏覽器擴展干擾
**症狀：** 無痕模式可以載入，正常模式不行

**解決方案：**
- 使用無痕模式（`Ctrl+Shift+N` 或 `Cmd+Shift+N`）
- 或逐一禁用瀏覽器擴展找出問題擴展

### 問題 4: 瀏覽器版本過舊
**症狀：** 控制台顯示語法錯誤

**解決方案：**
- 更新瀏覽器到最新版本
- 推薦使用：Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

## 📊 載入進度說明

系統會顯示三個步驟：
1. **步驟 1/3: 載入 Pyodide 腳本** - 從 CDN 下載腳本文件
2. **步驟 2/3: 初始化 Pyodide 環境** - 下載並初始化 Python 運行時（約 10-20 MB）
3. **步驟 3/3: 設置 Python 環境** - 配置執行環境

**正常載入時間：** 30-60 秒（首次載入）
**後續載入：** 5-10 秒（瀏覽器緩存）

## 🆘 如果所有方法都無效

### 檢查系統要求
- ✅ 現代瀏覽器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- ✅ 穩定的網絡連接（建議至少 1 Mbps）
- ✅ 足夠的內存（Pyodide 需要約 100-200 MB）

## 💡 預防措施

1. **首次使用時請耐心等待** - Pyodide 首次載入需要下載約 10-20 MB
2. **使用穩定的網絡** - 避免在網絡不穩定時使用
3. **保持瀏覽器更新** - 確保使用最新版本的瀏覽器
